## 1. Introduction to C++

- C++ is a general-purpose programming language.
- Developed by Bjarne Stroustrup in 1979.
- Extension of the C programming language.
- Supports:
  - Procedural programming
  - Object-Oriented Programming (OOP)
  - Generic programming

Common uses:
- System programming
- Game development
- Embedded systems
- High-performance applications

---

## 2. Structure of a C++ Program

Basic structure:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!";
    return 0;
}
```

Explanation:
- `#include <iostream>` → Library for input/output
- `using namespace std;` → Allows use of standard library names
- `main()` → Entry point of program
- `return 0;` → Program ended successfully

## 3. Variables and Data Types

### Basic Data Types

| Type      | Description |
|-----------|-------------|
| int       | Integer numbers |
| float     | Decimal numbers |
| double    | Double precision decimal |
| char      | Single character |

```cpp

float price = 19.99;
char grade = 'A';
bool isValid = true;
```

## 6. Control Statements

### If-Else

```cpp
if (condition) {
    // code
} else {
    // code
}
```

### Switch

```cpp
switch (choice) {
    case 1:
        cout << "One";
        break;
    default:
        cout << "Invalid";
}
```

### Loops

#### For Loop

```cpp
for (int i = 0; i < 5; i++) {
    cout << i;
}
```

#### While Loop

```cpp
while (condition) {
    // code
}
```

#### Do-While Loop

```cpp
do {
    // code
} while (condition);
```

## 7. Functions

Function Syntax:

```cpp
returnType functionName(parameters) {
    // code
    return value;
}
```

Example:

```cpp
int add(int a, int b) {
    return a + b;
}
```

Calling a function:

```cpp
int result = add(5, 3);
```

```cpp
int arr[5] = {1, 2, 3, 4, 5};
```

## 9. Pointers

Pointer declaration:

```cpp
int x = 10;
int* ptr = &x;
```

- `&` → Address of operator
- `*` → Dereference operator

Example:

```cpp
cout << *ptr;  // Outputs 10
```

## 10. Object-Oriented Programming (OOP)

### Class and Object

```cpp
class Person {
public:
    string name;
    int age;

    void greet() {
        cout << "Hello";
    }
};
```

Create object:

```cpp
Person p1;
p1.name = "John";
p1.greet();
```

### Key OOP Concepts

- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

---

## 11. Constructors

Constructor is automatically called when object is created.

```cpp
class Person {
public:
    string name;

    Person(string n) {
        name = n;
    }
};
```
